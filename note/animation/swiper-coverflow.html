Swiper Coverflow Effect

개요

Swiper의 coverflowEffect를 이용하여 슬라이드에 3D 입체 효과를 적용할 수 있습니다.

주요 내용

  • 입체적인 coverflow 효과로 슬라이드 강조
  • 현재 위치를 숫자로 표시하는 fraction 페이지네이션

사용한 예제

예제 코드


const swiper = new Swiper(".wrap", {
  direction: "horizontal",
  slidesPerView: "auto",
  spaceBetween: 0,
  centeredSlides: true,
  grabCursor: true,
  loop: true,
  speed: 1000,
  effect: "coverflow",
  coverflowEffect: {
    rotate: 50,
    stretch: -100,
    depth: 400,
    modifier: 1,
    slideShadows: false,
  },
  pagination: {
    el: ".swiper-pagination",
    type: "fraction",
  },
  navigation: {
    nextEl: ".swiper-button-next",
    prevEl: ".swiper-button-prev",
  },
  autoplay: {
    delay: 1000,
    disableOnInteraction: true,
  },
});

사용한 속성 설명

  • effect: 슬라이드가 전환될 때 어떤 시각 효과로 보여줄지 설정하는 속성으로, "coverflow" 설정으로 3D로 회전하며 중앙 슬라이드 강조 (입체적 효과)
  • coverflowEffect: effect: "coverflow" 일 때, 입체 회전 효과의 세부 옵션을 설정하는 객체
  • coverflowEffect.rotate: 슬라이드 회전 각도 (값이 클수록 회전 효과가 강해짐)
  • coverflowEffect.stretch: 슬라이드 간격 조절 (양수: 벌어짐 / 음수: 붙음)
  • coverflowEffect.depth: 슬라이드의 깊이(거리) → 값이 클수록 입체 효과가 큼
  • coverflowEffect.modifier: 전체 효과 강도 비율 (값이 클수록 효과가 더 강조됨)
  • coverflowEffect.slideShadow: 슬라이드 그림자 표시 여부
  • pagination.type: 페이지네이션 타입 속성으로, "fraction" 설정으로 숫자 형태로 표시

나의 생각 및 응용

Swiper.js의 coverflow 효과를 활용해 일반적인 슬라이드가 아닌 입체감 있는 전환 효과를 구현할 수 있었습니다.
기본 슬라이드보다 훨씬 시각적으로 흥미롭고 동적인 느낌을 줄 수 있어 사용자의 주목도를 높이는 데 효과적이라고 생각합니다.

다양한 옵션 값들을 바꿔보면서 슬라이드가 어느 정도 기울어지거나 겹쳐 보이는 느낌을 직접 조절할 수 있었는데, 이런 부분이 재밌었습니다.

앞으로 사용자의 시선을 끌고 싶은 영역에 이런 효과를 응용해서 다양한 스타일을 만들어보면 좋을 것 같습니다.

기술 적용 사례

이 기술은 Swiper 페이지 프로젝트의 이미지 슬라이드 구현에 활용했습니다.
