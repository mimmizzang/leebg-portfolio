Swiper Slide

개요

Swiper는 터치 슬라이드 기능을 제공하는 오픈 소스 라이브러리로, 웹 및 모바일 환경에서 부드러운 슬라이드 UI를 손쉽게 구현할 수 있습니다.

주요 내용

  • 가로 방향 이미지 슬라이드 구현
  • 자동 재생(autoplay), 반복 재생(loop) 설정
  • 슬라이드 이동 버튼(navigation), 페이지네이션(pagination) 추가

사용한 예제

예제 코드


var swiper = new Swiper("section .mySwiper", {
  slidesPerView: 3,
  spaceBetween: 10,
  centeredSlides: true,
  loop: true,
  pagination: {
    el: "section .swiper-pagination",
    clickable: true,
  },
  navigation: {
    nextEl: "section .swiper-button-next",
    prevEl: "section .swiper-button-prev",
  },
  autoplay: {
    delay: 3000,
    disableOnInteraction: true,
  },
});

사용한 속성 설명

  • slidesPerView: 한 화면에 표시할 슬라이드 개수
  • spaceBetween: 슬라이드 사이 간격 (px)
  • centeredSlides: 현재 활성화된 슬라이드가 중앙에 위치하도록 설정
  • loop: 슬라이드가 끝나면 처음으로 돌아가면서 반복 재생
  • pagination.clickable: 페이지네이션(하단 점)을 클릭해서 슬라이드 이동 가능
  • navigation.nextEl / prevEl: 이전/다음 버튼 설정
  • autoplay.delay: 슬라이드가 자동으로 전환되는 시간(ms) 설정
  • autoplay.disableOnInteraction: 사용자가 버튼 클릭 등으로 슬라이드 조작 시 자동재생 유지 여부

나의 생각 및 응용

이미지가 자동으로 부드럽게 슬라이드 되는 효과를 구현하기 위해 Swiper.js를 활용했습니다.
기본 기능 외에도 다양한 옵션들이 내장되어 있어서 복잡한 기능도 원하는 형태로 쉽게 커스터마이징할 수 있었습니다.

자동으로 넘어가면서 무한 반복되는 기능과 버튼이나 페이지네이션을 추가해서 원하는 슬라이드로 바로 이동할 수 있는 부분도 편리했습니다.

앞으로도 배너나 프로모션처럼 반복적으로 보여줘야 하는 콘텐츠에는 계속 활용하게 될 것 같고, 다른 프로젝트에서도 다양한 형태로 적용해보면 재밌을 것 같습니다.

기술 적용 사례

이 기술은 스타벅스 페이지 프로젝트의 프로모션 이미지 슬라이드 구현에 활용했습니다.
