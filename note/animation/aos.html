AOS Zoom

개요

AOS는 스크롤할 때 요소에 다양한 애니메이션 효과를 쉽게 추가할 수 있게 해주는 라이브러리입니다.
이 예제에서는 실제 프로젝트에 사용한 zoom 효과 위주로 정리했습니다.

주요 내용

  • 스크롤 위치에 따라 요소가 확대되며 등장하는 효과
  • data-aos="zoom-in-left" 또는 data-aos="zoom-in-right" HTML 속성으로 간단 적용
  • 추가 속성으로 애니메이션 지연 시간, 실행 횟수 등 설정 가능

사용한 예제

예제 코드


<section>
  <div class="inner">
    <div class="con" data-aos="zoom-in-left" data-aos-once="true">
      <div class="item">
        <img src="..." alt="" />
      </div>
    </div>

    <div class="con" data-aos="zoom-in-right" data-aos-once="true">
      <div class="item">
        <img src="..." alt="" />
      </div>
    </div>

    ...
  </div>
</section>

사용한 속성 설명

  • data-aos: 사용할 애니메이션 이름 지정
  • data-aos-once: true 설정 시 한 번만 애니메이션 실행

주로 사용하는 속성

  • data-aos-delay: 애니메이션 시작까지 대기 시간 (ms)
  • data-aos-duration: 애니메이션 실행 시간 (ms)
  • data-aos-offset: 화면에 보이기 시작하는 트리거 지점 간격 설정 (px)
  • data-aos-easing: 애니메이션 가속도 형태
  • data-aos-anchor: 애니메이션 트리거 기준 요소를 선택자로 지정
  • data-aos-anchor-placement: 트리거 요소의 어느 지점이 뷰포트에 닿을 때 애니메이션을 실행할지 설정

나의 생각 및 응용

스크롤할 때 화면에 자연스럽게 나타나는 애니메이션을 간단하게 구현하고 싶어서 AOS를 사용했습니다.
자바스크립트 코드를 몰라도 data-aos 속성만 추가하면 손쉽게 다양한 효과를 적용할 수 있어 학습 초기에도 부담 없이 사용할 수 있는 라이브러리라고 생각합니다.

더 세밀한 애니메이션이 필요하거나 커스텀 효과가 필요할 때는 AOS 기본 효과 외에도 직접 CSS 애니메이션이나 다른 JS 애니메이션 라이브러리와 함께 사용하면 좋을 것 같습니다.
