GSAP Floating 애니메이션

개요

GSAP을 이용하여 이미지나 아이콘이 위아래로 부드럽게 움직이는 애니메이션을 구현했습니다.
주로 시각적 강조가 필요한 요소에 사용되며, 단조로운 화면에 동적인 느낌을 더해줄 때 효과적입니다.

주요 내용

  • 요소 위아래로 움직이는 애니메이션
  • 애니메이션 랜덤 반복과 왕복 설정
  • GSAP의 gsap.to와 gsap.utils의 속성 사용

사용한 예제

예제 코드


gsap.utils.toArray(".floating-img").forEach((el) => {
  gsap.to(el, {
    y: gsap.utils.random(10, 30),
    duration: gsap.utils.random(1.5, 3),
    repeat: -1,
    yoyo: true,
    delay: gsap.utils.random(0, 1.5),
  });
});

사용한 속성 설명

  • y: 요소를 y축(세로 방향)으로 이동시키는 속성으로, 위아래로 자연스럽게 떠 있는 느낌을 표현할 때 사용
  • duration: 애니메이션이 한 번 재생되는 데 걸리는 시간(초) 으로, 각 요소마다 랜덤하게 지정하여 자연스러운 동작 연출
  • repeat: 애니메이션 반복 횟수 설정, -1로 설정하여 무한 반복
  • yoyo: 애니메이션이 끝난 후 반대로 재생하여 앞뒤로 왕복하는 동작을 만듦
  • delay: 애니메이션 시작 전 지연 시간을 랜덤하게 설정하여 여러 요소가 동일한 타이밍으로 움직이지 않도록 처리
  • gsap.utils.toArray(): 선택자 결과를 배열로 반환하여 여러 요소를 반복 처리할 때 사용
  • gsap.utils.random(): 지정한 범위 내에서 랜덤한 숫자를 반환하여 애니메이션에 변화를 줌

주로 사용하는 속성

  • opacity, scale, rotation: 요소의 투명도 변경, 크기 확대/축소, 회전
  • ease: 애니메이션 속도 곡선 설정
  • gsap.utils.wrap(array): 배열 내 값을 순환하여 반환
  • gsap.utils.clamp(min, max): 값이 범위를 넘지 않도록 제한

나의 생각 및 응용

이미지가 위아래로 움직이며 자연스럽게 떠다니는 효과를 구현하기위해 GSAP의 기본 애니메이션과 gsap.utils를 활용했습니다.
단순히 transform 속성을 이용하는 것보다 다양한 옵션과 랜덤 효과를 쉽게 적용할 수 있어 선택했습니다.

특히 gsap.utils.random()과 toArray()를 조합하여 여러 요소를 랜덤하게 처리함으로써, 화면에 규칙적이지 않은 자연스러운 움직임을 연출할 수 있었고,
시각적으로 지루하지 않은 효과를 줄 수 있었습니다.

이번 예제에서는 이미지에만 적용했지만, 버튼, 아이콘, 배경 요소 등 다양한 UI에도 활용할 수 있을 것으로 생각됩니다.
