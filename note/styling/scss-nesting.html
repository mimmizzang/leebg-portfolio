SCSS Nesting

개요

Sass의 유용한 기능중 하나는 중첩(Nesting) 기능입니다.
상위 선택자를 반복해서 쓰지않고 좀 더 편리하게 복잡한 구조를 작성할 수 있습니다.

주요 내용

  • 중첩(Nesting)
  • Ampersand(&)
  • @at-root

SCSS 코드


.footer_text {
  margin-top: 30px;
  text-align: center;

  span {
    font-size: 12px;
    color: gray;
    margin-right: 25px;

    &:last-child {
      margin-right: 0;
    }
  }
}

CSS 코드


.footer_text {
  margin-top: 30px;
  text-align: center;
}

.footer_text span {
  font-size: 12px;
  color: gray;
  margin-right: 25px;
}

.footer_text span:last-child {
  margin-right: 0;
}

속성 설명

  • 중첩(Nesting): 반복되는 상위 선택자를 생략하고, 계층 구조에 따라 중괄호 {} 안에 하위 선택자를 중첩해 작성할 수 있는 기능
  • Ampersand(&): & 키워드가 참조한 상위 선택자로 치환되는 기능

SCSS 코드


.message {
  color: black;

  @at-root .global-message {
    color: red;
    font-weight: bold;
  }
}

CSS 코드


.message {
  color: black;
}

.global-message {
  color: red;
  font-weight: bold;
}

속성 설명

  • @at-root: 중첩 구조에서 상위로 빠져나가는 기능
  • 조건부 스타일, 구조적 관리, 컴파일 제어 등 여러 경우에도 효과적인 기능
  • 코드는 지역적으로 위치, 결과는 전역으로

나의 생각 및 응용

SCSS의 중첩 기능을 활용하니 반복 코드를 줄일 수 있어 훨씬 깔끔하고 작성이 편했습니다.
특히 상위 선택자가 복잡할 때 코드 가독성이 좋아지고, 재사용할 때도 효율적이라는 걸 느꼈습니다.

실제 프로젝트에서는 주로 &와 일반 중첩 정도만 사용했지만, 공식 문서를 참고해 @at-root도 함께 실습하며 구조에 대한 이해도를 높일 수 있었습니다.

기술 적용 사례

이 기술은 스타벅스 페이지 프로젝트의 SCSS파일에 활용했습니다.
