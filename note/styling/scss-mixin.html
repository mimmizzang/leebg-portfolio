SCSS @mixin

개요

SCSS의 @mixin은 자주 사용하는 스타일 블록을 재사용 가능하게 만드는 기능입니다.
마치 함수처럼 스타일을 정의하고, 필요한 곳에서 @include로 호출해서 사용하는 방식입니다.

주요 내용

  • @mixin
  • @include

SCSS 코드


$main-black: rgb(53, 52, 52);
$deep-brown: rgb(94, 30, 30);
@mixin button-style(
  $color,
  $bg: transparent,
  $hover-text: white,
  $hover-bg: $color
) {
  color: $color;
  background-color: $bg;
  border: 2px solid $color;
  &:hover {
    background-color: $hover-bg;
    color: $hover-text;
  }
}
.btn {
  @include button-style($main-black);
}
.title {
  .btn {
    @include button-style($deep-brown);
  }
}

CSS 코드


.btn {
  color: rgb(53, 52, 52);
  background-color: transparent;
  border: 2px solid rgb(53, 52, 52);
}

.btn:hover {
  background-color: rgb(53, 52, 52);
  color: white;
}

.title .btn {
  color: rgb(94, 30, 30);
  background-color: transparent;
  border: 2px solid rgb(94, 30, 30);
}

.title .btn:hover {
  background-color: rgb(94, 30, 30);
  color: white;
}

속성 설명

  • @mixin 지시어를 이용하여 스타일을 정의합니다.
  • @include 지시어를 이용하여 선언된 mixin을 사용합니다.
  • mixin은 선택자를 포함 가능하고 상위(부모) 요소 참조(&)도 할 수 있습니다.
  • mixin은 함수처럼 인수를 가질 수 있습니다.
  • 인수는 기본값을 가질 수 있습니다. @include 포함 단계에서 별도의 인수가 전달되지 않으면 기본값이 사용됩니다.

나의 생각 및 응용

SCSS의 @mixin을 사용하니 반복되는 스타일을 줄이고, 코드가 훨씬 간결해졌습니다.

매개변수로 스타일을 조절할 수 있어 다양한 경우에 쉽게 재사용할 수 있었고, 공통적으로 적용되는 스타일이 많은 프로젝트일수록 mixin은 꼭 쓰게 되는 기능인 것 같습니다.

기술 적용 사례

이 기술은 스타벅스 페이지 프로젝트의 SCSS파일에 활용했습니다.
