CSS FadeIn

개요

fade-in 효과란 요소가 점점 불투명해지면서 부드럽게 나타나는 시각적 효과입니다.
CSS의 @keyframes와 animation 속성을 사용하여 부드럽게 나타나는 fade-in 효과를 구현했습니다.

주요 내용

  • @keyframes를 이용해 투명도 변화를 정의
  • animation 속성으로 애니메이션 적용

사용한 예제

예제 코드


@keyframes fadeIn {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

section .inner img {
  opacity: 0;
  animation-name: fadeIn;
  animation-duration: 0.5s;
  animation-timing-function: ease-in;
  animation-fill-mode: forwards;
}

.cup1 {
  animation-delay: 0.5s;
}
.cup2 {
  animation-delay: 1s;
}
.spoon {
  animation-delay: 1.5s;
}

사용한 속성 설명

  • @keyframes: 애니메이션 동작을 정의하는 규칙
  • animation-name: 적용할 애니메이션 이름 설정
  • animation-duration: 애니메이션 실행 시간 (초)
  • animation-timing-function: 애니메이션 속도 곡선
  • animation-fill-mode: 애니메이션 종료 후 적용 상태 설정
  • animation-delay: 애니메이션 시작까지의 지연 시간

주로 사용하는 속성

  • animation-iteration-count: 애니메이션 반복 횟수
  • animation-direction: 애니메이션 반복 시 방향
  • animation-play-state: 애니메이션 실행 상태 제어

나의 생각 및 응용

CSS만으로도 이미지가 하나씩 자연스럽게 등장하는 fadeIn 효과를 쉽게 구현할 수 있었습니다.

하지만 더 복잡한 애니메이션 조합이나 스크롤 연동 효과 같은 건 GSAP같은 라이브러리를 사용하면 더 정교하게 만들 수 있을 것 같습니다.

페이지가 열릴 때 집중도를 높이고 자연스러운 흐름을 만들어 주는 효과라 생각해서 앞으로 다용한 곳에 응용해보고 싶습니다.

기술 적용 사례

이 기술은 스타벅스 페이지 프로젝트의 이미지 fadeIn 효과 구현에 활용했습니다.
